# Описание

Ипподром — это консольное приложение на C++, позволяющее проходить аутентификацию и авторизацию и выполнять операции согласно варианту задачи. Проект состоит из нескольких файлов с исходным кодом. Структура проекта соответствует модели CIS, содержит папки bin, build, include, src. Документация проекта, включая постановку задачи и другие материалы, представленные в вики проекта. Проиллюстрированы различные типы запросов к базе данных, данные хранятся в различных форматах, включая изображения.
